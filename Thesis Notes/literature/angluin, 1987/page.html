Angluin, Dana. "Learning regular sets from queries and counterexamples." Information and computation 75.2 (1987): 87-106.

A minimally adequate teacher is assumed to answer correctly two types of questions from the learner about the unknown regular set.
  • 1. Membership Queries
  •   whether a string is a member of the unknown set or not.
  • 2. Conjecture (=vermoeden)
  •   whether a description of a regular set S, yes if S is equal to the unknown language or is a counterexample in the symmetric difference of S ad the unknown language.
Algorithm: L*
Let the unknown regular set be denoted by U and assume that it is over a fixed known finite alphabet A.

L* has always information about a finite collection of strings over A, classifying them as members or nonmembers of U. This can be organized into an observation table containing three strings:
  • 1. Non-empty finite prefix-closed (prefix-closed= each prefix of each member is also a member of the set) set S of strings
  • 2. Non-empty finite suffix-closed (suffix-closed= each suffix of each member is also a member of the set)  set E of strings
  • 3. Finite function T mapping ((S || S*A) *E) to {0,1}. , is 1 iff u is a member of U. Initially S=E={labda}
  • -> Observation table is denoted (S,E,T)
